What every license includes

The entire pipeline, on your hardware

Works offline

The PII detection sidecar runs locally. If you disconnect the laptop from the network, intake, testing, interviews, and audit all keep working. Only remote AI inference requires network, and PII is redacted before any remote call.

Your data, your disk

Case files live in a workspace folder on the clinician's machine. Nothing is stored on a Psygil server because there is no Psygil server for case data. The license server only verifies license keys.

Bring your own model key

Psygil uses the clinician's own Anthropic, OpenAI, or Azure OpenAI key for the Writer, Editor, Ingestor, and Diagnostician agents. API costs are billed to the clinician by the model provider, not by Psygil.

FAQ

Common questions

Is this a subscription?

No. A Psygil license is a one-time perpetual license for the version you purchased. You keep running that version as long as you want. Major version upgrades are offered as a separate upgrade purchase at a discount for existing license holders.

What happens after I pay?

Stripe emails you a receipt immediately. Within one business day, support@psygil.com sends your license key, download links for macOS, Windows, and Linux installers, and a short activation guide. Activate the app with your key on first launch. Each seat can be activated on one workstation at a time and reassigned if needed.

Which payment methods work?

All major credit and debit cards through Stripe. For ACH, wire transfer, purchase orders, or non-card payment, write to sales@psygil.com and we will send an invoice.

Does Psygil store my case files?

No. Case files are stored in a workspace folder on your machine. The Psygil license server verifies keys and does not receive or store any case data. PII detection runs locally in a sidecar process.

What about AI API costs?

The AI agents use your own Anthropic, OpenAI, or Azure OpenAI API key. You pay the model provider directly for inference. A typical full evaluation runs well under one dollar in API charges depending on the provider and model you select.

Can I try it before buying?

Write to sales@psygil.com with a short note about your practice. We offer a time-limited evaluation license for qualified forensic and clinical psychologists, and for referring counsel.

HIPAA and BAAs

Psygil is a desktop tool. Case data stays on the clinician's device. HIPAA obligations fall on the covered entity or business associate operating the device. For enterprise deployments that require a BAA with Foundry SMB, contact sales. Your model provider (Anthropic, OpenAI, Azure) is a separate BAA relationship you manage with them.

Refunds

Licenses are refundable within 30 days of purchase if the software does not work as described. Write to support@psygil.com with your license key and we will process the refund.

Which operating systems are supported?

macOS 12 or later (Apple Silicon and Intel), Windows 10 and 11, and most modern Linux distributions with glibc 2.31 or newer. See the download page for specific installers.
